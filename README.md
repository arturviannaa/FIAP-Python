## 📖 Sobre o Repositório

Este projeto tem como objetivo documentar o aprendizado progressivo de lógica de programação e pensamento computacional utilizando a linguagem Python. Aqui você encontrará desde conceitos básicos até estruturas lógicas mais complexas, organizadas de acordo com as aulas ministradas.

## 🗂️ Estrutura do Conteúdo

Os materiais estão divididos nos seguintes módulos/aulas:

- **📁 Aula 2 - Variáveis e Tipos de Dados:** Conceitos iniciais de alocação de memória, declaração de variáveis, manipulação de strings, inteiros, floats e booleanos.
- **📁 Aula 3 - Operadores:** Utilização de operadores matemáticos, relacionais e lógicos para a construção de expressões.
- **📁 Aula 4 - Condicionais:** Estruturas de controle de fluxo (`if`, `elif`, `else`) para tomada de decisão e validações lógicas dentro dos algoritmos.

*(Novas pastas serão adicionadas conforme o progresso das aulas).*
